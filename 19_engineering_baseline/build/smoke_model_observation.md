# 0.5B 冒烟模型实测记录

测试时间：2026-07-10

模型：`qwen2.5:0.5b`

第一轮四个 EOIR 正向样例结果：`0/4` 通过。

观察到的问题：

- 基线样例两次修复后仍有空脚本，未通过结构校验。
- 参数样例生成了单引号 AFSIM 调用，并错误地让 event pipe 使用 `.evt` 文件，`mission.exe` 返回 1。
- 双航点样例保留了参数，但生成脚本不符合 AFSIM 语法，`mission.exe` 返回 1。
- 三窗口样例只生成一组动作，没有满足三组 `BeginImaging`/`EndImaging`。
- 增加命令正则约束后，0.5B 单次生成超过 64 秒仍未返回，测试被人工终止；评测器随后增加了每样例 90 秒硬超时。

结论：0.5B 可以证明 Ollama 和本地 API 可用，但不能作为本项目的工程生产模型。完整工程结论必须等待 `qwen2.5:7b` 对最新五样例评测集的结果。
