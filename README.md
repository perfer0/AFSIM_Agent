# AFSIM Agent Learning Project

这个目录按学习步骤组织，每一步一个独立文件夹。

## 目录

```text
01_json_to_afsim_main/   第一步：JSON -> AFSIM 主控脚本 -> mission.exe 验证
```

## 学习顺序

先进入 `01_json_to_afsim_main`，看里面的 `README.md`。

后续每一步都会新建文件夹，例如：

```text
02_generate_platform_instances/
03_component_library/
04_local_llm_json_generation/
05_rag_knowledge_base/
06_mcp_tools/
07_agent_skills/
```
